\# 🚀 M5+ | Personal Portfolio Architecture



A premium, high-performance developer portfolio engineered with \*\*Vanilla Web Technologies\*\* and optimized for seamless User Experience (UX), Performance, and Internationalization (i18n).



![Portfolio Preview](./assets/images/og-preview.webp)



\## 🧠 The "Systems Thinker" Approach

This project avoids heavy frameworks for a simple portfolio, proving that robust architecture, modularity, and advanced features can be achieved beautifully with Vanilla HTML, CSS, and JavaScript.



\## ⚙️ Core Architecture \& Features

\- \*\*Modular CSS Architecture:\*\* Split into isolated modules (`variables`, `reset`, `layout`, `components`) for zero CSS bloat and easy maintenance.

\- \*\*Dynamic i18n Localization:\*\* Fully bilingual (English/Arabic) with instantaneous switching via a Vanilla JS dictionary object and \*\*CSS Logical Properties\*\* for seamless LTR/RTL flipping.

\- \*\*Premium Animations:\*\* Custom highly-performant Vanilla JS `IntersectionObservers` for scroll reveals, and a lightweight 60fps 3D Tilt Hover effect.

\- \*\*Perfect Performance:\*\* 100/100 Lighthouse score achieved via automated `.webp` image conversion, strict lazy loading strategies, and asynchronous decoding.

\- \*\*Glassmorphism UI:\*\* Advanced backdrop-filters with dynamic Dark/Light theme variables.



\## 🛠️ Tech Stack

\- \*\*Frontend:\*\* Semantic HTML5, Modular CSS3, Vanilla JavaScript (ES6+).

\- \*\*Tooling:\*\* Node.js (Sharp) for automated image compression script.

\- \*\*Deployment:\*\* Netlify with advanced Open Graph (OG) \& SEO meta configurations.
